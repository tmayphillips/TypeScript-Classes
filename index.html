TypeScript Classes

A TypeScript class is declared with the class keyword followed by the class name.
Inside the class we can define variables, constructors, and functions. 


              class Planet {
                listOfPlanets:Planet[] = []
                name:string
              
                constructor(name:string) {
                  this.name = name
                }
          
                displayPlanet():void {
                  { add planet to html page }
                }
              }

Inheritance

A subclass inherits all of the properties and methods of its parent class.


            class Planet {
              listOfPlanets:Planet[] = []
              name:string
            
              constructor(name:string) {
                this.name = name
            }

            class GasGiant extends Planet { 
              gases:string[]
              constructor(name:string) {
                  super(name) 
                  this.gases = gases
            }

Visibility

Public

Public is the default visibility for class members, and they can be accessed from anywhere.
There is no need to write public, but can add it for readability. 


              class Planet {
                displayPlanet():void {
                  { add planet to html page }
                }
              }

              displayPlanet()

Protected

Protected members are only visible to the subclasses of their class. 


              class Planet {
                protected displayPlanet():void {
                  { add planet to html page }
                }
              }

              class IceGiant extends Planet {
                public display() {
                  this.displayPlanet()
                }
              }

Private

Private members can only be accessed within the same class. 


              class Planet {
                displayPlanet():void {
                  { add planet to html page }
                }

                displayPlanet()
              }

Static

Static members are not associated with any particular instance of a class.
They can be accessed using dot notation. They can be public, private, or protected. 


              class Planet {
                public static listOfPlanets:Planet[] = []
                private name:string
              }

              newPlanet = new Planet(name);
              Planet.listOfPlanets.push(newPlanet);

Getters

A getter is also called an accessor.
A getter method returns the value of the property's value. 


              class Planet {
                protected displayPlanet(): void {
                  let fluids = planet).fluids
                }
                
              }
              class IceGiant extends Planet {
                fluidElements:string[];
                constructor(fluidElements:string[]) {
                    super() 
                    this.fluidElements = fluidElements;
                }
            
                public get fluids():string{
                    return this.fluidElements.join(' , ')
                }
            }

A setter is also called a mutator.
A setter method updates the property's value.

Chaining


              class numbOp {
                private n:number;
                constructor(num:number) {
                    this.n = num;
                }
            
                public add(inc:number = 1):this
                    this.n = this.n + inc;
                    return this;
                }
                toString() { 
                    return this.n;
                }
              }

              let finalNumber = new numbOp(3).add().add(2).toString();
              console.log(finalNumber) // Output: "6"
